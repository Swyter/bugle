NAME

stats_fragments - gather fragment counts  

SYNOPSIS

filterset stats_fragments
 

DESCRIPTION

The stats_fragment filter-set provides signals for the number of fragments to other filter-sets. It must be listed in a chain that uses any statistics related to fragment counts.

It is simply a wrapper around GL_ARB_occlusion_query, so a ``fragment'' is defined exactly as in that extension (and OpenGL 1.5), and will not work unless that extension is supported.  

BUGS

Fragment counting will not work in applications that use occlusion queries. A warning is displayed when this condition is detected.
