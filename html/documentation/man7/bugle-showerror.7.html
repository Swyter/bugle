NAME

showerror - reports errors in OpenGL calls  

SYNOPSIS

filterset showerror
 

DESCRIPTION

This filter-set detects OpenGL errors and writes them to the log (see bugle-log(7)). With the default log settings, the error will be reported to standard error.  

AUTHOR

gldb and the bugle library are written and maintained by Bruce Merry <bmerry@users.sourceforge.net>.
